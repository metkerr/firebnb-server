## **Tech Stack**

- NodeJS
- ExpressJS
- EJS

## **Usage**

### Install dependencies

```
npm install
```

### Run Server (Visit http://localhost:3001)

```
npm start
```

### Run Server (DEV Mode) (Visit http://localhost:3001)

```
npm run dev
```
